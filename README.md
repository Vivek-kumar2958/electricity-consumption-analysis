# Electricity Consumption Analysis

## Project Overview

This project analyzes electricity consumption patterns across different states and regions of India using Tableau.

The goal is to understand electricity usage trends, compare regional consumption, and study the impact of lockdown on electricity usage.

---

## Dataset

The dataset used in this project is **Consumption.csv**.

It contains the following fields:

* State
* Region
* Latitude
* Longitude
* Date
* Electricity Usage
* Usage Category

---

## Tools Used

* Tableau Public
* SQL
* HTML
* CSV Dataset

---

## Visualizations Created

The following visualizations were developed:

1. State Consumption (2019)
2. State Consumption (2020)
3. Region Wise Usage
4. Monthly Electricity Consumption
5. Quarter Usage
6. Yearly Usage
7. Top and Bottom N States
8. Lockdown Impact Analysis

---

## Dashboard

The Tableau dashboard combines multiple charts to provide an overview of electricity consumption trends including:

* Region wise consumption
* Monthly electricity usage
* Lockdown impact on electricity usage
